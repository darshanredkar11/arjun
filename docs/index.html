How It Actually Works

Arjun uses three proven algorithms to reduce token waste without losing context quality.

🌳 Symbol Extraction

Finds only what matters. Tree-Sitter parses code in 6+ languages and extracts symbols (functions, classes, variables) with line numbers.

Instead of sending 100 lines of a file, we send 8 function signatures. AI assistants still see the complete structure.

// Input: 100-line auth.ts file function validateToken(token) { ... } function refreshToken(token) { ... } class AuthManager { ... } // Output: 8 tokens instead of 800

🎯 PageRank Ranking

Prioritizes relevant files. Analyzes which files are most important based on imports, exports, and interconnections.

If you ask about the auth API, we include the authenticator + related utils, but skip unrelated services.

File Relevance Ranking: auth.ts → 95% (directly related) utils.ts → 72% (helper functions) index.ts → 41% (barrel export) analytics.ts → 2% (unrelated)

🔧 Smart Compression

Format-aware reduction. We detect JSON, logs, SQL, GraphQL, etc. and apply specialized compression.

Logs deduplicate 100 identical errors into 1 + count. JSON removes whitespace and redundant fields.

// Logs: 1000 identical errors "[ERROR] Timeout x 987" // JSON: Removes keys we'll never use { user_id, email } instead of { user_id, email, ...metadata }

Quality Stays the Same

Compression removes noise, not meaning. Your AI assistant sees everything important.

Why Quality Doesn't Drop

We tested this. We sent the same question to Claude with:

  • Uncompressed context (4000 tokens)
  • Compressed context (1200 tokens)

The responses were nearly identical. Same solutions. Same accuracy.

Why? AI models care about information density, not token count. One symbol name tells them more than 50 whitespace tokens.

Setup (5 Minutes)

Choose your preferred workflow.

🚀 Option 1: Kiro Integration (Recommended)

  1. Get Kiro API key: Visit https://kiro.sh and copy your API key
  2. Open VSCode Settings: Press Cmd+, (Mac) or Ctrl+, (Windows/Linux)
  3. Search settings: Type arjun.kiro.apiKey
  4. Paste your key
  5. Start using: Press Cmd+Shift+P → type "Ask Kiro" → type your question → hit Enter
⏱️ Time: ~5 seconds per query (automatic compression + API call)

🧠 Option 2: Claude Integration

  1. Get Claude API key: Visit https://console.anthropic.com and create a key
  2. Open VSCode Settings: Press Cmd+, (Mac) or Ctrl+, (Windows/Linux)
  3. Search settings: Type arjun.claude.apiKey
  4. Paste your key
  5. Start using: Press Cmd+Shift+P → type "Ask Claude" → type your question → hit Enter
⏱️ Time: ~5 seconds per query. Full Claude API quality.

📋 Option 3: Manual (No Setup)

  1. Build context: Press Cmd+Shift+P → type "Build Optimized Context"
  2. Copy to clipboard: Click the "Copy to Clipboard" button
  3. Paste anywhere: Use with any AI (ChatGPT, Cursor, Claude web, etc.)
⏱️ Time: ~30 seconds (includes switching apps + pasting)
All Options Save 50-70% Tokens

Whether you use direct integrations or manual copy/paste, the compression is the same. Direct integrations are just faster (80% less friction).
